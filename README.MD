# confiabilitips — Proyecto HTML básico

Resumen
-------
Sitio estático mínimo llamado "confiabilitips" con una página principal ([blog.html](blog.html)). Proyecto para almacenar contenido estático e imágenes.

Estructura del repositorio
--------------------------
- [.gitignore](.gitignore) — reglas para ignorar archivos/directorios.
- [blog.html](blog.html) — página principal del sitio.
- [notas.txt](notas.txt) — notas breves del proyecto.
- [imagenes/](imagenes/) — carpeta prevista para imágenes.
- [fotos/](fotos/) — carpeta prevista para fotos.

Cómo ver el proyecto localmente
------------------------------
1. Abrir [blog.html](blog.html) en un navegador o ejecutar un servidor estático en la carpeta del proyecto:
```sh
python -m http.server 8000
# luego abrir http://localhost:8000/blog.html
```
